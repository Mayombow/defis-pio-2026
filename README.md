# defis-pio-2026

## 💎 GEM RUSH — jeu 2D multijoueur (3–4 joueurs)

Un jeu d'arène 2D ultra-coloré jouable à **3-4 en ligne** depuis le navigateur, en
**peer-to-peer (WebRTC)**, sans installation ni serveur — idéal pour un hébergement
**GitHub Pages**. Sécurités réseau intégrées (reconnexion auto, heartbeat, reprise de
score…).

👉 **Le jeu et sa documentation sont dans [`game/`](game/).**
Une fois GitHub Pages activé, il est accessible à `…github.io/<repo>/game/`.

> L'app « Camp Pio — Expédition » (`index.html`) reste inchangée à la racine.
